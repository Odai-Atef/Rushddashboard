# Implementation Plan: Project Dashboard Summary API Integration

**Branch**: `059-project-dashboard-summary` | **Date**: 2026-06-21 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/059-project-dashboard-summary/spec.md`

## Summary

Replace all hardcoded dashboard data in `ProjectDashboardPage.tsx` with live data fetched from `GET /api/v1/projects/dashboard/summary`. Introduce a reusable `useProjectDashboard` hook in `src/api/hooks/` that encapsulates the API call, loading state, and Arabic error messaging. Render stats, status distribution pie chart, budget trend line chart, recent activity feed, and upcoming deadlines directly from the API response. Remove the static `project-data.ts` file once no other page imports it and the build passes.
